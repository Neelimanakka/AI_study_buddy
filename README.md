# 📚 AI Study Buddy
AI Study Buddy is an AI-powered web application designed to help students learn topics easily through AI-generated explanations, technical practice questions, voice interaction, and multilingual support. It provides an interactive and user-friendly learning experience for students preparing for exams and improving their knowledge.
# 🌐 Live Demo
**Website:**  
https://ai-study-buddy-1-06qv.onrender.com/
# ✨ Features
- 📖 AI-generated explanations for any study topic
- 📝 Generates 20 important technical practice questions with answers
- 🌍 Supports multiple languages
  - English
  - Telugu
  - Hindi
  - Tamil
  - Kannada
- 🎤 Voice input using Speech Recognition
- 🔊 Voice output using Speech Synthesis
- 💾 Save study progress
- 📊 View study history
- 📱 Responsive design for desktop and mobile devices

# ⚙️ Installation
### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/AI_study_buddy.git
```
### 2. Open the Project Folder
```bash
cd AI_study_buddy
'``
### 3. Install Required Packages
```bash
pip install -r requirements.txt
```
### 4. Create a `.env` File
```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
```
### 5. Run the Application
```bash
python app.py
```
Open your browser and visit:
```
http://127.0.0.1:5000
```
# 🚀 How It Works
1. Enter your name.
2. Select your preferred language.
3. Enter the topic you want to study.
4. Click **Explain** to receive an AI-generated explanation.
5. Click **Practice (20 Qs)** to generate important technical questions with answers.
6. Use **Speak Topic** to enter topics using your voice.
7. Listen to explanations using voice output.
8. Save your study progress.
9. View your previous study history anytime.
